Frequently Asked Questions

What is PixlCheck and what does it do? +

PixlCheck is an AI-powered web tool that analyzes images to determine whether they are real photographs or AI-generated visuals using deep learning techniques.

How do I check if an image is AI-generated? +

Simply upload an image using the upload box on the homepage. PixlCheck instantly processes the image and displays whether it is AI-generated or real.

Does PixlCheck work on mobile devices? +

Yes. PixlCheck is fully responsive and works seamlessly on desktops, tablets, and smartphones without requiring any installation.

What image formats are supported? +

PixlCheck currently supports JPG and PNG image formats with a maximum upload size of 8MB per image.

Are uploaded images stored on your servers? +

No. PixlCheck respects user privacy. Uploaded images are processed temporarily and are not stored after analysis.

Can PixlCheck be integrated into other platforms? +

API integration is planned for future releases, enabling developers and organizations to integrate PixlCheck into their workflows.
